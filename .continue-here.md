## What Was Done This Session

### Bugs Fixed (PDF loading chain ‚Äî 4 issues)
All four were blocking the PDF from rendering at all:

1. **CSP blocked blob: worker** ‚Äî `index.html` was missing `worker-src 'self' blob:`; PDF.js worker URL was rejected
2. **Worker missing `Uint8Array.toHex` / `fromBase64`** ‚Äî Chrome 129+ APIs absent in worker's isolated JS global; fixed by blob wrapper that installs polyfills before importing real worker
3. **`Map.prototype.getOrInsertComputed` missing** ‚Äî TC39 upsert proposal, Chrome 136+; Electron 35 ships Chromium 134; polyfilled in both renderer and worker contexts
4. **`page.render()` deprecated API** ‚Äî `usePdfRenderer.ts` was passing `canvasContext`; pdfjs-dist 5.5.x requires `canvas` directly

Plus: **unused import TypeScript error** in `useViewportControls.ts` (`MIN_ZOOM`, `MAX_ZOOM`) was blocking Vite from starting ‚Äî removed.

### Files Modified
- `src/renderer/index.html` ‚Äî `worker-src 'self' blob:` added to CSP
- `src/renderer/src/lib/pdf-setup.ts` ‚Äî blob wrapper with polyfills; workerSrc points at blob
- `src/renderer/src/hooks/usePdfRenderer.ts` ‚Äî `page.render({ canvas })` instead of `canvasContext`
- `src/renderer/src/hooks/useViewportControls.ts` ‚Äî removed unused imports
