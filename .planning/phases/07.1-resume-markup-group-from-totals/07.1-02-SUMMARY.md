# Phase 7.1 Plan 02: TotalsRow Extensions Summary

TotalsRow extended with exported helpers, onArmTool callback, and a 20px hover-only Plus slot that arms the matching markup tool without reflowing the UoM column.

## What Was Built

**Task 1 — Exported helpers:**
- `labelToName(label: string): string` — strips disambiguation suffix `\s*\((count|linear|area|perimeter|wall)\)\s*$`; used by App.tsx `handleArmTool` to get the clean item name
- `rowTypeToMarkupType(t: BoqRowType): MarkupType` — maps `perimeter-length` and `perimeter-area` → `perimeter`; all others pass through; used by App.tsx to derive the markup tool type from the BOQ row

**Task 2 — Three changes:**
1. `TotalsRowProps` extended with `onArmTool?: (item: BoqItemRow) => void` (after `onTriggerPulse?`)
2. `handleClick` fires `onArmTool?.(item)` in the zero-match early-return path (before `return`) AND at the end of the normal cycle navigation path
3. 20px fixed-width Plus slot added after the UoM `<span>`:
   - Always 20px wide regardless of hover state — no UoM column reflow
   - `Plus size={14}` icon renders only when `bg !== 'transparent'` (row is hovered)
   - `title="Add more to this group"` tooltip
   - No `onClick` handler — click propagates to row `handleClick`
   - Icon color: `COLORS.textSecondary` (unobtrusive)
