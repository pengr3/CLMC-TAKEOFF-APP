  <how-to-verify>
Run the app: `npm run dev`

**Scenario A — Live totals (VIEW-01 success criterion 1):**
1. Open a project with markups on a calibrated page
2. Switch to the Count tool, place 3 new count pins named "Light Switch"
3. Observe the TotalsPanel right side — "Light Switch" row quantity should update immediately after each pin placement (no delay, no export needed)
4. Undo the last placement (Ctrl+Z) — quantity decrements immediately
PASS: Quantities update within one render frame of placing or removing a markup.

**Scenario B — Thumbnail navigation (PDF-05):**
1. Open a multi-page PDF (≥3 pages) with markups on multiple pages
2. Click thumbnail tile page 2 — canvas jumps to page 2; active-page outline (2px accent border) moves to tile 2
3. Click thumbnail tile page 3 — canvas jumps to page 3; outline moves
4. Verify markup overlay on thumbnails matches the markups visible on the canvas for each page
PASS: Click navigates correctly; active tile highlighted; thumbnail overlay matches canvas.

**Scenario C — 1080p layout (ROADMAP success criterion 3):**
1. Resize app window to 1920×1080 (or set to full-screen on a 1080p display)
2. Both panels open at default widths (ThumbnailStrip 140px, TotalsPanel 320px)
3. Canvas center column must be clearly usable — plan is readable, not squashed
4. Drag TotalsPanel splitter to 400px — canvas shrinks but remains usable
PASS: Canvas occupies majority of horizontal space with default panel widths. Plan is readable.

**Scenario D — Pulse highlight + HoverRing zoom fidelity (D-11/D-12):**
1. Open a project with count pins and linear markups
2. Navigate TotalsPanel to a page with both types
3. Hover a count-pin item row — HoverRing should appear as a steady white ring around matching pins (at current zoom)
4. Zoom in to 4x — hover again — ring should remain same visual size (zoom-compensated)
5. Click the row — PulseHighlight should appear as a colored ring/outline, fade out over ~1.5 seconds
6. Mentally count "one-thousand, two-thousand" — pulse should be fully gone by "two-thousand"
PASS: Ring anchors to markup geometry at all zoom levels; pulse fades in approximately 1.5s.

**Scenario E — Panel persistence across reload (D-02/D-03):**
1. Collapse the ThumbnailStrip (click chevron)
2. Drag TotalsPanel to 400px width
3. Close and reopen the app (Ctrl+Q or close window, npm run dev again)
4. ThumbnailStrip should be collapsed; TotalsPanel should be 400px
PASS: Both panel states persist across app restart.

**Scenario F — "Copy as text" (D-14):**
1. Right-click any item row in TotalsPanel
2. Context menu appears with "Item" section header and "Copy as text" action
3. Click "Copy as text"
4. Paste into a text editor — should be tab-separated: `{label}\t{quantity}\t{uom}`
5. Observe "Copied {label}" confirmation toast appears and fades
PASS: Clipboard receives correct tab-separated payload; toast confirms.
