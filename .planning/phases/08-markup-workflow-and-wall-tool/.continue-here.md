All 20 numbered decisions live in `08-CONTEXT.md` (D-01 through D-20). Highlights:

**Chain mode**
- Break trigger: Esc OR re-click active tool button (tool-switch implicitly breaks)
- Visual: existing toolbar active style + name+color chip badge near button
- Cross-page: chain persists across page navigation
- Storage: runtime-only, inside `useMarkupTool`

**Wall tool**
- New `'wall'` MarkupType (separate shape) with `points` + `wallHeight` (mm)
- Per-wall popup input on commit; chain inherits last wall's height; first wall default = 2400mm
- Reuse `MarkupNamePopup` with conditional `Wall height: [___] mm` row when `toolType === 'wall'`
- BOQ representation: single row per name in m² (length × height), matches area markups
- Edit support extends Phase 7's `EditMarkupCommand`

**Show/hide visibility**
- Per-project (additive `hiddenItemNames: string[]` in `.clmc`)
- Lucide `Lightbulb` / `LightbulbOff`, left of color chip in TotalsRow, always visible
- Canvas hidden; row stays interactive; totals & BOQ export include hidden items
- NOT undoable

**Crosshair**
- Rifle-scope style (1px crossed lines + circular center gap, no dot, white with thin black outline)
- CSS cursor with SVG data-URL on `CanvasViewport` `containerRef`
- Active for all markup tools + Set Scale + Verify Scale
