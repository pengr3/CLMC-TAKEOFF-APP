## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 2 - Missing Critical] React 19 incompatible renderHook import**
- **Found during:** Task 5 (v1-migration-pdfbytes.test.ts)
- **Issue:** Plan specified `import { renderHook, act } from 'react'` but `renderHook` is not exported from `react` in React 19 without `@testing-library/react`. Tests would permanently fail with `TypeError: renderHook is not a function` even after Wave 4 implements the target symbols.
- **Fix:** Used `createRoot + act` component-mounting pattern consistent with existing test patterns in the codebase (title-bar-dirty.test.ts, toolbar-open-prop.test.ts). Wrapped hook call in a `TestComponent` that captures results via `useEffect + Promise`. Same pattern applied in replace-plan-pdf.test.ts.
- **Files modified:** src/tests/v1-migration-pdfbytes.test.ts, src/tests/replace-plan-pdf.test.ts
- **Verification:** Tests now fail with proper assertion errors (expected vs received) rather than import errors
- **Committed in:** 3e2e8e6 and 179e1cb
