## IPC Contract After This Plan

### file:readProject

```typescript
// Channel: 'file:readProject'
// Input: filePath: string
// Returns: ReadProjectResult (discriminated union)
type ReadProjectResult =
  | { kind: 'v2-zip'; projectJson: string; pdfBytes: Uint8Array; computedSha256: string }
  | { kind: 'v1-json'; text: string }
  | { kind: 'unknown'; reason: string }
```

- v2-zip: ZIP extracted, pdfBytes returned as Uint8Array, computedSha256 is 64-char hex (main hashed)
- v1-json: raw JSON text returned; renderer migrates + marks dirty (Wave 4)
- unknown: corrupt/empty file; renderer surfaces OpenErrorModal

### file:writeProject

```typescript
// Channel: 'file:writeProject'
// Input: (filePath: string, jsonText: string, pdfBytes: Uint8Array)
// Returns: { ok: true }
```

Atomic write sequence:
1. `assembleClmcZip(jsonText, pdfBuf)` — produce ZIP Buffer
2. `writeFile(${path}.tmp, zipBuf)` — write to temp
3. `rename(${path}.tmp, ${path})` — atomic promotion
4. On rename failure: `unlink(${path}.tmp)` — cleanup orphan, rethrow

### file:hashBuffer

```typescript
// Channel: 'file:hashBuffer'
// Input: bytes: Uint8Array
// Returns: string (64-char lowercase hex SHA256)
```

Renderer calls this before writeProject to populate `project.json.pdf.sha256`.
