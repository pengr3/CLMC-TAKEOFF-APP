<objective>
Replace the Phase 11 tab-switcher inside ScalePopup with a pre-choice gate dialog (ScaleMethodDialog).

Purpose: The tab-switcher that Phase 11 added inside ScalePopup created a UX problem — the user was forced into drawing mode before choosing a scale method. The correct flow is: click "Set Scale" → see a compact choice dialog → pick "Draw line" OR "Type ratio 1:N" → proceed into the chosen method without any prior canvas interaction.

Output:
- CalibMode gains a 'pre-choice' state
- New ScaleMethodDialog component handles both the choice view and the ratio entry view
- ScalePopup reverted to draw-line confirm flow only (tab bar removed)
- CanvasViewport wired to show ScaleMethodDialog at pre-choice, ScalePopup at confirming only
</objective>
